Quick Start

Get started with a simple query

curl 
# Simple Query
curl -X POST http://localhost:50051/db.v1.DatabaseService/Query \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "database": "primary",
    "sql": "SELECT * FROM users LIMIT 10"
  }'

# Begin Transaction
curl -X POST http://localhost:50051/db.v1.DatabaseService/BeginTransaction \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"database": "primary", "timeout": "30s"}'

API Endpoints

All endpoints use Connect/gRPC protocol over HTTP/2

POST /db.v1.DatabaseService/Query Execute a simple query
STREAM /db.v1.DatabaseService/QueryStream Stream large result sets
POST /db.v1.DatabaseService/BeginTransaction Start a new transaction
POST /db.v1.DatabaseService/TransactionQuery Execute query within transaction
POST /db.v1.DatabaseService/CommitTransaction Commit the transaction
POST /db.v1.DatabaseService/RollbackTransaction Rollback the transaction
STREAM /db.v1.DatabaseService/InteractiveTransaction Bidirectional streaming transaction
POST /db.v1.DatabaseService/ExecuteTransaction Execute atomic transaction script
POST /db.v1.DatabaseService/GetDatabaseSchema Get full database schema
POST /db.v1.DatabaseService/Explain Explain query plan
POST /db.v1.DatabaseService/Vacuum Run VACUUM command
POST /db.v1.AdminService/Login Authenticate and get API key
